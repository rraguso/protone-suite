The following table lists all default shortcut keys used by the ProTONE Player application.
TIP:  To view and change the actual shortcuts used by the application, use the Keyboard panel from Settings dialog

Command name Description Shortcut Alternative shortcut Configurable
OpenHelp Open help file F1 F1 No
ShowLogConsole Displays the log file console F12 F12 No
Play Play file. Z MediaPlayPause Yes
Pause Pause playback. X Space Yes
Stop Stop playback. C MediaStop Yes
Prev Move to previous file in playlist. V MediaPreviousTrack Yes
Next Move to next file in playlist. B MediaNextTrack Yes
Load Load new file(s). N SelectMedia Yes
OpenDisk Opens the Load DVD media dialog Ctrl+D Ctrl+D Yes
OpenURL Open a URL Ctrl+U Ctrl+U Yes
FullScreen Toggle Full Screen Alt+Enter Ctrl+F Yes
Fwd Fast forward with 5 seconds. Right OemPeriod Yes
Rew Rewind with 5 seconds. Left Oemcomma Yes
VolUp Volume up by 5%. Add VolumeUp Yes
VolDn Volume down by 5%. Subtract VolumeDown Yes
MoveUp Move selected items with one position up in playlist. PgUp NumPad9 Yes
MoveDown Move selected items with one position down in playlist. PgDn NumPad3 Yes
Delete Delete selected items. Del Decimal Yes
Clear Delete entire playlist. Ctrl+Del Ctrl+Decimal Yes
LoadPlaylist Load a new playlist. Ctrl+N Ctrl+SelectMedia Yes
SavePlaylist Save the current playlist. Ctrl+S Ctrl+S Yes
LoopPlay Toggles playing in loop. Ctrl+L Ctrl+L Yes
PlaylistEnd Activates the 'action on playlist end' Ctrl+E Ctrl+E Yes
ToggleShuffle Activates playlist shuffle mode. Ctrl+H Ctrl+H Yes
JumpToItem Open the 'Jump to playlist item' dialog Ctrl+J Ctrl+J Yes
CfgVideo Displays Video Configuration Ctrl+Alt+V Ctrl+Alt+V Yes
CfgAudio Displays Audio Configuration Ctrl+Alt+A Ctrl+Alt+A Yes
CfgSubtitles Displays Subtitles and OSD Configuration Ctrl+Alt+T Ctrl+Alt+T Yes
CfgTimer Displays Timer Options Ctrl+Alt+S Ctrl+Alt+S Yes
CfgRemote Displays Remote Control Configuration Ctrl+Alt+R Ctrl+Alt+R Yes
SearchSubtitles Searches subtitles for movie clips Ctrl+T Ctrl+T Yes
BookmarkManager Open bookmark manager Ctrl+Alt+B Ctrl+Alt+B Yes
OpenSettings Displays the Settings dialog. Ctrl+Alt+C Ctrl+Alt+C Yes
CfgKeyboard Displays the Keyboard Shortcuts. Ctrl+Alt+K Ctrl+Alt+K Yes
