This dialog is displayed whenever the user presses the "Show log file" shortcut key (F12). Typically, this window is brought up in the process of troubleshooting the application.

The user can type fragments of the playlist element name he is trying to locate, and the list will show the matching playlist elements.
By double clicking an element in the list, the application will begin playing the repective playlist element.

If the user presses Enter, the application will begin playing the selected element from the list.
