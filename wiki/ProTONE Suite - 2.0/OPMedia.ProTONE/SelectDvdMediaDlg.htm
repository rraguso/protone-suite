This dialog helps the user to choose the DVD media to load in the application.
Typically, it is brought up when the user presses the "Open DVD" button from the Playback Panel or the related shortcut (Ctrl + D)

To open a DVD disk, select it from the list box and press OK.
You can also load DVD media copied locally: use Browse button then locate the folder that contains the DVD media.

TIP:A DVD media folder is either a VIDEO_TS folder or its parent.
